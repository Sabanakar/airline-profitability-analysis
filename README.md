# Airline Profitability Analysis

 Project Overview
This project analyzes airline route profitability using operational, revenue, and cost data from 7,900+ flights across 30 routes.

The goal was to identify:
- The most profitable and least profitable routes
- Key cost drivers impacting profitability
- Revenue vs cost structure differences
- Monthly profit trends
- Strategic improvement opportunities


Executive Dashboard
The project includes an executive-level performance dashboard featuring:

- Top & Bottom 5 Routes by Profit Margin
- Profit Contribution by Route Category
- Revenue vs Total Cost Comparison
- Monthly Profit Trend Analysis
- Top 5 Cost Drivers Impacting Profit


Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook


 Key Insights
- Long-haul routes generate significantly higher margins.
- Certain cost categories (Fuel, Marketing, IT Systems) have the strongest correlation with profit.
- Profitability shows clear seasonal fluctuations.
- Revenue growth does not always guarantee margin improvement.
